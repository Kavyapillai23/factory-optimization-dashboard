# 🏭 Factory Optimization Dashboard

An interactive **AI-powered Streamlit dashboard** designed to optimize supply chain and factory allocation decisions through data-driven insights.

---

## 🚀 Key Features

### 📊 1. KPI Dashboard
- Lead Time Reduction
- Profit Impact Stability
- Scenario Confidence Score
- Recommendation Coverage

### 🚀 2. Factory Optimization Simulator
- Compare predicted performance across factories
- Identify best factory based on lead time

### 🔄 3. What-If Scenario Analysis
- Simulate trade-offs between **speed vs profit**
- Visualize impact on lead time and profitability
- Supports decision-making with dynamic inputs

### 📊 4. Deep Insights
- Units vs Lead Time analysis
- Lead Time distribution
- Ship Mode performance comparison

### 📝 5. Recommendation Engine
- Suggests optimal factory allocation
- Highlights key operational bottlenecks

### ⚠️ 6. Risk & Impact Panel
- Identifies high-risk assignments
- Provides risk distribution insights
- Alerts for critical operational issues

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Pandas & NumPy**
- **Plotly**
- **Scikit-learn (ML Model)**

---

## 🎯 Business Use Case

This dashboard helps:
- Optimize factory allocation
- Reduce delivery lead time
- Balance profit vs speed trade-offs
- Identify operational risks proactively
